Instalación
===========

## Instalación como dependencia de otro proyecto

```
{
    "require": {
        "mostofreddy/loggy": "1.0.*"
    }
}
```

Luego ejecutar `composer install` o `composer update`.


## Instalacion para desarrollo

```
git clone git@github.com:mostofreddy/logger.git
cd logger
composer install
```
